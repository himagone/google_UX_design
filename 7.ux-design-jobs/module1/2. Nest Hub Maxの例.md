# ソーシャルグッドの事例
インクルーシブな視点で製品デザインに取り組むことの重要性を繰り返し強調してきました。  
アイデア出しの段階で、社会的に弱い立場にあるグループが見過ごされることが多いという事実を認識しているからこそ、  
優れたUXデザインは、できるだけ多くのユーザーを包摂するユーザー体験の創出を目指すべきだということを、ぜひ理解していただきたいと思います。  
高齢者介護施設の入居者のニーズに焦点を当てることで、Nest Hub Maxの改善につながったソーシャルグッドのためのデザインの事例を紹介します。

## Nest Hub Max
Nest Hub Maxは、ビデオ通話やタスクのスケジュール管理、その他のスマート技術機能を提供するスクリーンデバイスです。  
視覚障害や聴覚障害のある人々のためのアクセシビリティオプションを含むプロトタイプの作成は、今では一般的になっています。  
例えば、GoogleのNest Hub Maxが登場したとき、タッチ機能付きのHDスクリーンを搭載していました。
ユーザーは、より使いやすくするために、テキストサイズや音量などの使いやすさの設定をカスタマイズできました。 
### 発生した問題
問題は何だったのでしょうか?アクセシビリティを考慮して作られているものの、　　
製品自体は健常者を中心に設計されていたのです。　　
障害のある人々が自分に合わせて製品を調整することはできますが、製品は彼らのニーズに特化して作られたわけではありませんでした。

### 対象となるユーザーグループの発見
しかし2020年、Nestチームは方向転換し、こうしたユーザーグループの1つに焦点を当てることにしました。　　
GoogleのHardware for Goodイニシアチブの一環として、　　
チームは介護施設の入居者にとって、より繋がりのある生活を実現することを目指しました。　　
世界中で何百万人もの高齢者が介護施設で暮らしています。
この人口層は、外の世界から感情的にも社会的にも孤立していると感じることがよくあります。　　
家族から遠く離れた場所にいる入居者もいれば、深刻な健康問題につながる病気にかかりやすいため、厳しい面会規制下に置かれている入居者もいます。　　
Nestチームは、Nest Hub Maxを作成する際に、聴覚や視覚の低下といったアクセシビリティの懸念を考慮していましたが、　　
高齢者施設に住む人々のニーズを満たすことはできませんでした。

### 問題の解決
解決策として、Nestチームは、ワシントン州の施設を皮切りに、　　
高齢者施設の参加入居者向けにカスタマイズされたNest Hub Maxスマートディスプレイをセットアップするプログラムを立ち上げました。　　
まず、デザイナーたちは、テクノロジーの操作に慣れていない人々の間でつながりを促進するアクションを簡素化しました。  
画面上のカードやプロンプトを通じてアクティビティへの入り口を増やすことで、  
入居者はNest Hub Maxディスプレイを効率的に操作できるようになりました。  
また、このデザインにより、連絡先へのアクセスやビデオ通話がより簡単にできるようになりました。  
次に、チームはタッチではなく音声操作を重視した体験をデザインしました。  
高齢者は皮膚の水分が少ないことが多いため、タッチスクリーンがタップを検出しにくく、  
皮膚の導電性に関するタスクが問題になることがあります。  
音声操作機能を作ることで、デバイスと物理的にやり取りする必要がなくなります。
最後に、Nestチームはプラットフォームのセットアップを簡単にすることで、Nest Hub Maxを改善しました。  
サインインにGoogleアカウントを必要とする代わりに、  
プラットフォームはサインアウトモードで動作し、最小限のデータのみを収集します。  
これにより、テクノロジーに詳しくない入居者でも、専門知識なしにプラグ&プレイで簡単に使えるようになりました。

### 問題に対応するためのアップーデート
これらのNest Hub Maxのアップデートは、高齢者施設の入居者という特定のオーディエンスを念頭に置いて作られました。　　
しかし、これらの新機能は、より幅広いオーディエンスのアクセシビリティも向上させることになります。　　
例えば、シンプル化されたデザインは、タスクを素早くスケジュールしたい忙しいビジネスパーソンにも適しています。　　
また、音声コマンドは、料理をしながらビデオ通話をするなど、複数のアクティビティを同時にこなそうとするユーザーにも役立ちます。　　
社会的に弱い立場にあるグループの視点からUXプロセスを見直すことで、　　
結果的にあらゆる種類のユーザーを助けることになったのです。　　
これから自分の製品をデザインする際は、特定の集団にとって有益なものが、　　
より広いオーディエンスにとっても同じように役立つことが多いということを覚えておいてください。
